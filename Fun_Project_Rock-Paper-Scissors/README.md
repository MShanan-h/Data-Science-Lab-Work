# Rock-Paper-Scissors Game

## Description

This project is a simple implementation of the Rock-Paper-Scissors game in Python. The user plays against the computer, which randomly selects its move. The program determines the winner based on standard game rules.

## Dataset

* Description: This project does not use a dataset. It generates random choices using Python.

## Steps Performed

* Model Building: Implemented game logic using conditional statements

## Results

* The game successfully allows the user to play multiple rounds against the computer.
* The program keeps track of scores and displays the final result.

## Tools Used

* Python
* random module

## Conclusion

This project demonstrates the use of basic Python concepts such as loops, conditional statements, and random number generation to build an interactive game.
